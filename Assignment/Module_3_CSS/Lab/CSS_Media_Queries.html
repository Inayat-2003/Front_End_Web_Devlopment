HTML

The HTML uses a long and complex doctype declaration, making it harder to remember.

It includes outdated elements like font, center, and big.

Generally it requires external plugins like Flash or Silverlight for audio and video.

HTML has limited input types, requiring JavaScript for form validation.

HTML5

The HTML5 simplifies it to !DOCTYPE html, making it easier to use.

It removes these, promoting CSS for styling and layout.

The HTML5 introduces audio and video elements for native media support.

HTML5 adds new input types like email, date, and number, along with attributes like placeholder and required.
